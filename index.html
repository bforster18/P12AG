POWER BI E-FUEL PROJECT

This user guide introduces the Power BI E-Fuel Project dashboard. It helps users explore global e-fuel projects and analyse key metrics such as CO₂ capture, hydrogen production, e-fuel production, power sources, project status, company involvement, project insight, assumptions, and key definitions.

Page 1
Map of E-Fuel Projects
Explore the global project landscape and filter by company, project, region, fuel type, country, status and operational year.

What you see

This page displays an interactive world map showing all E-Fuel projects included in the dataset. Each point on the map represents a project. The color of each point reflects the selected legend, such as electrolyzer technology.

Available filters
  • Company: Displays projects associated with a specific company.
  • Project: Allows selection of individual projects.
  • Super region: Filters projects by geographic region (Europe, Asia, Africa, etc.).
  • E-Fuel: Filters by the type of fuel (e-ammonia, e-diesel, e-kerosene, etc.).
  • Country: Restricts the view to one or multiple countries.
  • Status: Filters by project status (announced, under construction, operational, cancelled, etc.).
  • Operational year: Filters by the year of the start of plant operation.
How to use this page
  • Click on geographic areas to zoom in.
  • Use the filters to focus on specific companies, fuels, regions, or technologies.
  • Use the legend buttons to switch between E-Fuel type, CO₂ source, Electrolyzer technology, Power source and Project status.
  • The total number of projects updates automatically based on the selected filters.
  • To get insights for a specific project, right-click on a project on the map, then click Drill through to open the Project Insight page.

Project Insight
Examine the full pathway from power generation to synthetic fuel production for a specific project.

Overview

This page focuses on Project Insight and allows users to examine the full pathway from power generation to synthetic fuel production for a specific project.

Available filter
  • To explore a project, start from the map page, right-click on a project, then select Drill through to open Project Insight with all available project information.
How to use this page
  • Use the Open Web Link button (Ctrl + Click) to access external references.
  • Use the map to confirm the geographic location of the project.
  • Power Source shows the renewable energy source used for the project (e.g., solar, wind).
  • Electrolysis displays electrolyzer technology, vendor, installed capacity, published capacity, and estimated capacity.
  • Hydrogen Production shows published hydrogen production and estimated hydrogen production.
  • CO₂ Capture provides information on capture technology, emission source, capture company, and captured CO₂ quantities.
  • E-Fuel Production displays the final synthetic fuel production pathway and quantities.
  • Blank fields indicate missing published data.
Data Consistency Note

For some projects, published values for H₂, CO₂, and E-Fuel production may be inconsistent.

Estimated values are calculated using stoichiometric coefficients for each fuel type, including ammonia, methanol, methane, and kerosene.

These estimations may lead to significant differences between published and estimated values.

Assumptions and Estimation Methodology
Understand how missing technical values are estimated to keep the E-Fuel project database consistent and comparable.

Overview

This section explains the methodology used to estimate missing technical values in the E-Fuel project database. Because project announcements often contain incomplete technical information, an estimation model was developed to derive consistent values for hydrogen production, CO₂ capture, and e-fuel output.

Hydrogen production estimation

Hydrogen production is estimated based on the installed electrolyzer capacity.

  • Assumption: 100% capacity factor
  • Operating time: 8,760 hours per year
  • This corresponds to the maximum theoretical annual production capacity.
Electrolyzer efficiency assumptions
  • SOEC: 50 kWh/kg H₂
  • AEM: 63 kWh/kg H₂
  • ALK: 64 kWh/kg H₂
  • PEM: 66.8 kWh/kg H₂
  • Unknown: 65 kWh/kg H₂

These factors allow conversion between electrical capacity and hydrogen production output.

Hydrogen to E-Fuel conversion

To estimate e-fuel production, the model uses stoichiometric conversion factors describing the relationship between hydrogen, CO₂, and the final fuel.

  • E-Methanol: H₂ to Fuel 5.298 · Fuel to H₂ 0.1889 · CO₂ to Fuel 0.728 · CO₂ to H₂ 0.259
  • E-Methane: H₂ to Fuel 1.990 · Fuel to H₂ 0.5025 · CO₂ to Fuel 0.365 · CO₂ to H₂ 0.183
  • E-Ammonia: H₂ to Fuel 5.632 · Fuel to H₂ 0.1776 · CO₂ to Fuel 0.000 · CO₂ to H₂ 0.000
  • E-Kerosene: H₂ to Fuel 2.284 · Fuel to H₂ 0.4380 · CO₂ to Fuel 0.323 · CO₂ to H₂ 0.141
  • E-Diesel: H₂ to Fuel 2.292 · Fuel to H₂ 0.4370 · CO₂ to Fuel 0.322 · CO₂ to H₂ 0.141

These coefficients allow estimation of hydrogen demand, CO₂ capture requirements, and final e-fuel production.

Dashboard assumptions
  • Projects operate at 100% of nominal capacity once operational.
  • Production remains constant after the operational start year.
  • Estimated values are used to fill missing data gaps.

These assumptions allow the dashboard to provide a complete and consistent view of global e-fuel production capacity over time.

Definitions
Review the definitions used across the dashboard for status, scale, position, electrolyser technology and CO₂ source.

Key definitions

This section gathers the main definitions used throughout the Power BI E-Fuel Project dashboard.

Status
  • Announced: The project has been publicly announced but has not yet entered detailed planning or development.
  • Cancelled: The project has been officially stopped and is no longer planned to be developed or implemented.
  • DEMO: The project is a demonstration facility intended to test, validate, or showcase a technology at a pilot or near-commercial scale.
  • End of Operation: The project or facility has permanently ceased operations.
  • FEED (Front-End Engineering Design): The project is in the detailed engineering phase where technical design, cost estimates, and project specifications are developed before a final investment decision.
  • Pre-FEED: The project is in an early engineering and feasibility assessment phase preceding FEED, where initial technical concepts and project options are evaluated.
  • FID (Final Investment Decision): The project has reached the stage where investors or project owners have formally approved the capital investment to proceed with construction.
  • Operational: The facility has been built and is currently in operation.
  • Under Construction: Construction of the facility has started but the project is not yet operational.
Scale
  • DEMO (Demonstration Plant): A demonstration-scale project designed to test, validate, and showcase technology before large-scale commercial deployment.
  • Industrial: A full-scale facility intended for commercial or industrial production rather than technology testing or validation.
Position
  • Position refers to location of the e-production site.
  • If the project location has not yet been specified by the company, the position refers to the company’s headquarters.
Electrolyser technology
  • ALK: Alkaline electrolysis
  • PEM: Proton exchange membrane electrolysis
  • SOEC: Solid oxide electrolysis cells
CO₂ source
  • Biogenic: CO₂ originating from biological sources such as biomass, biogas, or bio-based processes.
  • Direct Air Capture (DAC): CO₂ captured directly from the atmosphere using dedicated air capture technologies.
  • Fossil Power Generation: CO₂ captured from power plants that generate electricity using fossil fuels such as coal, oil, or natural gas.
  • Industrial: CO₂ captured from industrial processes such as cement, steel, chemical, or refining production.
